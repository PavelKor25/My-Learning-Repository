Перед повторением изучения проектов данной папки рекомендую начать с папки React projects (CodeSandbox). Здесь приводятся папки в локальной среде разработки, т е VSCode.

Вкратце, как создать проект React, с которым можно совершать удобную динамическую отладку с помощью редактируемой страницы в браузере (не без помощи инструмента Vite):

1) Сначала необходимо проверить наличие node.js и npm, проверив версии:

node -v
npm -v

Если чего-то из этого нету, необходимо установить node.js (как экзешник) (npm встроен в node, он установится вместе с node)

2) Зайти в папку с проектами, или в пустую папку нового проекта (на свой выбор):

cd "..."

3) Создать папку нового проекта с помощью React и Vite:

npm create vite@latest *app-name* --template react

Пояснение:
npm create создает что-то из npm;

vite@latest Создает Vite - инструмент разработки фронтенд-приложений. latest указывает на последнюю версию.

*app-name* - название папки приложения. По канону лучше писать через тире, но можно и с пробелами.

--template react - для Vite существуют различные начальные состояния проектов, в зависимости от указанного шаблона. Здесь параметр template указывает на React-структуру.

4) Перейти на созданную папку с помощью cd. (см. п. 2)
5) Последние две команды:

npm i
npm run dev

Первый устанавливает необходимые зависимости из указанных в package.json, а второй запускает сервер и Vite-React проект.
